# party-hard-restarter

This is a simple script for getting "Clubber" achievement for "[Party Hard](https://store.steampowered.com/app/356570/Party_Hard/)" (for starting game for hundered times).

### To run:
```
powershell -ExecutionPolicy Bypass -File .\party-hard-restarter.ps1
```
### To stop:
```
Just close the terminal window (or press CTRL+C)
```
